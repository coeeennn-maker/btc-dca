⚡ How This Works

Signal Logic

Open this app any day. If the composite signal is in buy/sell territory, you get an exact dollar amount. No fixed schedule — quality entries only.

Conservative Mode (2026)

Only buy below risk 0.30

Diminishing returns each cycle shrink the value zone. For 2026, only accumulate below 0.3. Toggle off for the full 0–0.5 range.

Kelly Criterion (Regime-Aware)

Position size is set by empirical half-Kelly fractions derived from backtesting 5,780 daily BTC data points at 90-day forward returns — separately calibrated for bull, bear, and accumulation regimes.

Five Signal Multipliers

  • MVRV proxy (price ÷ 2yr avg): <1.0 ×1.20 / <1.5 ×1.10 / <2.0 ×1.00 / <2.5 ×0.90 / <3.5 ×0.75 / >3.5 → sell
  • Halving cycle: early bull ×1.00 / bull peak ×0.80 / bear onset ×0.70 / deep bear ×0.90 / pre-halving ×1.10
  • Volatility adj: caps Kelly when 30d realized vol exceeds historical 65%
  • Fear & Greed: ≤25 ×1.15 / ≤45 ×1.08 / ≤55 ×1.00 / ≤75 ×0.92 / >75 ×0.85
  • Dip vs 30d MA: >10% below ×1.15 / 5–10% ×1.08 / near ×1.00 / surge ×0.85

Sell Ladder (Cowen + MVRV)

  • Risk 0.70–0.75 → sell 12%
  • Risk 0.75–0.85 → sell 25%
  • Risk 0.85–0.92 → sell 45%
  • Risk >0.92 → sell 70%
  • MVRV >3.5 (price 3.5× its 2yr avg) → sell 15%

Max 80% of position sold ever. Last 20% = forever stack.

Australian CGT — Lot Optimiser

Every sell signal ranks your specific lots by CGT efficiency: lots held 12+ months (50% discount) recommended first, then by highest cost basis to minimise taxable gain. Tax-loss harvest opportunities flagged automatically.

Data Storage

All purchases and sells are saved to your browser's local storage on this device. Use Export to back up your data as a JSON file. Import to restore it on a new device or browser.

Regime-Aware Spacing

Minimum days between buys scales with regime and risk: Bull 3d · Accum 5d · Bear <15% risk 7d · Bear 15–22% risk 14d.

Daily Alerts

Scheduled task checks every morning at 8am. Signal fires only when active. Silence = hold.
